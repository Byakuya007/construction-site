REMEMBER THE STORE YOU VISITED?
That's what aims to solve. With the help of this section, you can mark any location you visit on the map with a name of your choice and Done !
It doesn't matter if it's a big shop or a local shop. This acts like a memory for keeping track of the places you visited
anywhere in kenya so go on, give it a try..!

Steps to Use:

Step 1. Allow location access to the popup appeared above in the browser.

Step 2. Mark the desired location you like to save on the map.

Step 3. Move cursor over '+' icon and type a location name then press Enter.
